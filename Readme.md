IV6 VFD 6 digit 7 segment digital clock with Alarm function and Date, Humidity and Temperature display.

Microcontroller: STM32F030F4P6
 
RTC module:DS3231

Temperature and Humidity sensor: AM2302

1 button for change Clock, Date or Alarm settings.

2 button for value change plus or minus direction.

Developed in STM32cube IDE 2.0.0.
